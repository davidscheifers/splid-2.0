# splid-2.0

## Installation

1. install dependiencies

```bash
$ npm install
```

2. create .env file and add aws api key

```env
VITE_API_KEY=<your-api-key>
```

3. start development server

```bash
$ npm run dev
```

## Build project

```bash
$ npm run build (builds with typescript compiler)

or

$ npx vite build (for quick test build)
```
